Pittsburgh Crime Visualization Project

For my Data Visualization class at Penn State, I chose to work on mapping crime incidents reported in the Greater Pittsburgh area. For this project, I drew inspiration from a prior class assignment mapping State College crime data using the Leaflet JavaScript library, which can be found here. This webpage will provide a brief overview on the techniques and technologies used to accomplish the final product, shown in the video at the end of this page.

Data Collection, Cleaning, and Transforming

The data for this project was downloaded from the Western Pennsylvnia Regional Data Center (WPRDC) website. Two datasets were downloaded, historical and past 30 days. The data was in csv format, and loaded into R. To start, data was filtered to only include entries from 2022, blank rows and rows without location information were dropped, and column names were changed for a clearer table. After this, the two tables were appended together to get a full view of the data. The final step was creating logic to generalize category names. In the original data, there were many category descriptions describing the same type of incident, so generalizing these made it manageable to sort by category. The new data table was then exported to a csv file to use in Tableau. An R notebook with the code used for data processing can be found here

Tableau Visualization

After the data processing was complete, the new file was imported to a Tableau Workbook. The features of the data had to be set to the right format (e.g. Geolocation > Latitude for Latitude variable), then the map visualization was used. For a cleaner finish and greater interactivity, Mapbox was used to add an OpenStreetView map to the background, showing roads, town names, and giving users the ability to easily zoom and move the map. Two maps were created, one with an individual point for each incident, the other displaying a count of incidents, creating a density map. Interactive filters were added, allowing users to view only certain months, sort by category, or a combination of the two.
